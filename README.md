# Chamada Escolar

Sistema de controle de presença para professores.

## Acesso
- ADM: usuário `adm` / senha `admin123`
- Professor demo: `professor@demo.com` / senha `demo123`

## Funcionalidades
- Chamada com marcação de presença e falta
- Alerta automático de faltas
- Cadastro de alunos com observações
- Múltiplas turmas
- Painel do diretor com relatórios
- Controle de professores com senha
